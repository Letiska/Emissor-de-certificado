# Emissor de certificado automático

Um pequeno projeto em python  para emitir pdfs de certificados automáticamente a partir de modelos em pdf para servir de templete e listas de nomes em arquivos txt.
As dimensões do modelo, posição, tamanho, cor e fonte dos nomes são definidas no código.
Este projeto foi feito para emitir certificados de um congresso, mas pode ser adaptado para outras aplicações.
